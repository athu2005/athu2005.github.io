Battery RUL Prediction using EDA & ML

Machine Learning Analysis of Li-ion Battery Degradation for Remaining Useful Life Estimation

Overview

This course project under Prof. Alankar combined exploratory data analysis with machine learning to predict the remaining useful life (RUL) of lithium-ion batteries. The project analyzed over 15,000 cycling records from 14 different Li-ion cells to identify key degradation patterns and build predictive models.

Project Objectives

  • Analyze battery degradation patterns across multiple cells and cycling conditions
  • Identify key features correlating with battery health and remaining useful life
  • Develop machine learning models for accurate RUL prediction
  • Compare and optimize different modeling approaches
  • Provide actionable insights for battery management and replacement strategies

Data Analysis & Feature Engineering

Dataset Scale: 15,000+ battery cycles from 14 Li-ion cells

Tools Used: Python (Pandas, NumPy), Seaborn for visualization

Key Correlations Discovered

  • Discharge Time → RUL: Correlation coefficient of 0.78 (strong positive relationship)
  • Discharge Voltage → RUL: Correlation coefficient of ±0.76 (bidirectional patterns)
  • Cycle Count: Primary degradation indicator tracking battery age
  • Temperature Effects: Significant impact on degradation rate and predictions

Machine Learning Models Developed

1. XGBoost Model

Performance: RMSE = 67.1 cycles

  • Gradient boosting approach with ensemble learning
  • Effective at capturing non-linear degradation patterns
  • Feature importance analysis revealed discharge voltage as top predictor
  • Fast inference suitable for real-time battery management systems

2. LSTM (Long Short-Term Memory) Model

Performance: RMSE = 59.3 cycles (BEST)

  • Deep learning model capturing temporal degradation sequences
  • Superior performance over XGBoost by 11.6% in RMSE
  • Effectively learned long-term dependencies in battery cycling data
  • Better handling of variable-length sequences across different cells

Methodology

  • Data Preprocessing: Handled missing values, normalized features, created rolling window features
  • Train-Test Split: 80-20 split with temporal ordering preservation
  • Hyperparameter Tuning: Grid search and cross-validation for both models
  • Validation: Cross-validation with multiple metrics (RMSE, MAE, R²)

Key Findings

1. Battery degradation follows a predictable trajectory that can be accurately modeled with sufficient historical data

2. LSTM networks significantly outperform traditional ML for this temporal sequence prediction task

3. Discharge voltage is the most critical feature for RUL prediction, followed by cycle count

4. Temperature variations must be accounted for in reliable RUL predictions

Real-World Applications

  • Battery Management Systems: Predict maintenance needs before battery failure
  • Electric Vehicles: Estimate remaining range and battery health for users
  • Energy Storage Systems: Optimize charging strategies based on degradation predictions
  • Warranty Management: Data-driven warranty terms based on actual degradation patterns
  • Cost Optimization: Plan replacement schedules to minimize total cost of ownership

Model Deployment Considerations

The LSTM model achieved the best performance with RMSE of 59.3 cycles, translating to ±1.5% error in typical battery cycles. This accuracy enables reliable deployment in real-time battery management systems and IoT devices.
