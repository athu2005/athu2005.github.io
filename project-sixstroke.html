Six-Stroke Engine Cam Profile Design

Novel Camshaft System with MATLAB Simulation & ADAMS Dynamic Analysis

Overview

This advanced combustion engine design project developed a novel six-stroke camshaft system under Prof. Seshu. The project combined MATLAB-based profile generation with MSC ADAMS dynamic simulation to analyze a unique combustion cycle concept.

Innovation: Six-Stroke Concept

Traditional four-stroke engines follow intake-compression-power-exhaust cycles. This project introduced a six-stroke configuration to explore improved efficiency and emissions characteristics. The additional strokes target better scavenging and thermal optimization.

Technical Approach

  • Cam Profile Generation: Used MATLAB to create intake and exhaust cam profiles using raised cosine lift modeling
  • Profile Optimization: Optimized for smooth acceleration, minimal vibration, and efficient valve opening/closing
  • Dynamic Simulation: Exported profiles to MSC ADAMS for full dynamic simulation with contact modeling and friction analysis
  • Validation: Compared results with traditional four-stroke configurations to identify efficiency gains

Key Findings

Critical Discovery: The six-stroke configuration exhibited a 35% reduction in intake flow capacity compared to the equivalent four-stroke engine. This was an important finding that guided further optimization efforts toward compensatory measures like increased valve sizing and optimized timing.

MATLAB Implementation

Profile Modeling: Raised cosine lift curves for smooth valve transitions

Output: CAD-compatible profile data with parametric control of lift, duration, and timing

Visualization: Comprehensive plots showing velocity, acceleration, and jerk characteristics

ADAMS Simulation

Model Components: Camshaft, valve follower, springs, push rods, and valve head assembly

Analysis: Contact forces, friction losses, fatigue analysis, and resonance characteristics

Validation: Compared simulated results with experimental data from conventional engines

Lessons & Applications

  • Deep understanding of advanced combustion cycle concepts
  • Expertise in multi-body dynamics simulation and validation
  • Experience with parametric CAM design for engine optimization
  • Foundation for future work on efficiency improvements in ICEs (Internal Combustion Engines)

Future Work

The findings from this project contributed to the EcoCombust startup concept for a 50% efficiency improvement in mono-fuel six-stroke engines. The cam design methodologies developed here are directly applicable to that commercialization effort.
