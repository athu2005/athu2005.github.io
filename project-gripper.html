Robotic Gripper Arm

Servo-Actuated Gripper for Mobile Robot with Optimized Precision & Speed

Overview

This project involved designing and fabricating a servo-actuated robotic gripper arm for the SeDriCa mobile robot. The gripper was optimized through iterative design cycles and field testing to achieve fast, reliable object pickup with precision alignment.

Design Objectives

  • Fast actuation speed for time-constrained tasks
  • Strong grip force without damaging fragile objects
  • Precise alignment for accurate object positioning
  • Lightweight construction for minimal impact on mobile platform
  • Modular design for easy maintenance and upgrades

Key Design Features

Actuation System: High-torque servo motor with geared linkage mechanism for force multiplication

Mechanism: 4-bar linkage with spring-loaded fingers for adaptive gripping

Materials: Aluminum frame with 3D-printed finger pads for grip texture

Control: PWM servo control interfaced with ROS2 for autonomous operation

Development Process

  • Conceptualization: Analyzed gripper requirements and compared parallel-jaw vs. compliant gripper designs
  • CAD Design: Created detailed 3D model in SolidWorks with linkage analysis
  • Prototyping: Fabricated first prototype using 3D printing and machining
  • Field Testing: Iteratively tested with various objects and refined finger geometry, spring tension, and servo speed
  • Optimization: Final design achieved optimal balance between grip strength, speed, and weight

Performance Metrics

Grip Force: 50-100N adjustable force maintaining object integrity

Actuation Time: 0.5-1.2 seconds for full grasp

Weight: 350g total assembly

Positioning Accuracy: ±5mm alignment tolerance

Impact on Mobile Platform

The gripper significantly enhanced the SeDriCa robot's manipulation capabilities, enabling autonomous object pickup and placement tasks. The modular design allowed for easy integration with the robot's existing control system.

Future Improvements

  • Implement tactile feedback sensors for intelligent grip detection
  • Add suction cup alternative for smooth object pickup
  • Reduce weight further using carbon fiber materials
  • Develop machine learning model for object-specific grip optimization
