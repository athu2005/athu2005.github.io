Connected Mobility Testbed

Cyber-Physical Lab for Indian Traffic Conditions

Overview

Adapted an advanced Cyber-Physical Lab originally designed for urban European environments into a testbed optimized for Indian traffic dynamics. The solution implements real-scale 1:14 RC vehicle platform with sophisticated motion planning and Ackermann steering kinematics.

Technical Implementation

Vehicle Dynamics Architecture

Developed comprehensive kinematic model incorporating Ackermann steering geometry with precise steering angle computation. The platform supports dynamic trajectory tracking for complex Indian traffic patterns including aggressive lane changes, unpredictable pedestrian behavior, and variable road infrastructure.

ROS2 Integration

Architected modular ROS2 stack with decoupled perception, planning, and control subsystems. Node architecture enables scalable sensor fusion pipeline with real-time performance optimization for resource-constrained embedded platforms.

Motion Planning System

Implemented hybrid path planning combining graph-based methods for global optimization with local trajectory refinement for dynamic obstacle avoidance. System handles chaotic traffic scenarios through probabilistic collision prediction and reactive safety override protocols.

Results & Impact

  • Traffic Adaptation Success: Successfully validated motion planning in 50+ simulated Indian traffic scenarios
  • Real-time Performance: Achieved 100+ Hz control loop execution on ARM processors
  • Research Publication: Findings presented at Cyber-Physical Systems workshop
  • Platform Robustness: Extended testbed framework for multi-vehicle coordination research

Core Learnings

  • Kinematic modeling and steering control system design for scaled platforms
  • ROS2 middleware for modular robotic systems architecture
  • Motion planning algorithms adapted for chaotic traffic environments
  • Real-time system optimization for embedded robotic platforms
